# My THRIVE-Belize

### ▶ Open the app: **https://yeridu.github.io/thrive-belize/**

A student companion web app for the **THRIVE-Belize** life-skills program at Toledo Community
College (Punta Gorda, Toledo District, Belize). It gives students an easy way to review what
they learned across the **7 modules and 31 sessions**, look up key words, track their own
journey, and find help.

> **Looking for the app?** Click the link above. (This page only shows the source code;
> the running app lives at the GitHub Pages link.)

## What you can do

- Browse the 7 modules and all 31 sessions
- Read each session's key ideas, things to remember, key words, reflection, and practice-at-home
- Search across everything, and look up terms in the glossary
- Keep a private **Passport** to mark sessions reviewed and note what you learned
- Tap **Get Help** on any screen

## Good to know

- This is a **skills-program companion -- not counselling or therapy**.
- It works on a phone, works **offline**, and makes **no network requests**. Anything you write
  in the Passport is saved **only on your device** and is never uploaded or shared.

## For the THRIVE-Belize team

- One self-contained `index.html` (inline CSS/JS, no server, no build step). GitHub Pages serves
  it from the `main` branch root.
- Content is **student-facing only**, drawn from the manual's Student Toolkit handouts.
- **Before student use:** fill in the help-contact placeholder shown on the *Get Help* screen
  (`[TO BE COMPLETED BY THE THRIVE-BELIZE TEAM: verified school counsellor contact and national
  helpline numbers]`).
